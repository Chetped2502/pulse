# Pulse — Product Spec

## ภาพรวม
เว็บแอปรันในเครื่องส่วนตัว (local) เชื่อมต่อบัญชี TikTok ของเจ้าของระบบผ่าน
TikTok Official API (Login Kit + Display API) เพื่อติดตามผลงานคลิปของตัวเอง
ไม่มีฟีเจอร์ดูเทรนด์/คู่แข่ง เพราะ API ไม่รองรับ

## ข้อจำกัดที่ต้องรู้ (สำคัญ)
- กราฟแนวโน้ม (7/30 วัน) จะค่อยๆ มีข้อมูลตามจำนวนวันที่ sync จริง
  วันแรกที่เริ่มใช้ กราฟจะแทบว่างเปล่า
- ไม่มีข้อมูลเพลง/เสียงที่ใช้ในคลิป (TikTok API ไม่ให้ข้อมูลนี้)
- แฮชแท็กได้มาจากการตัดคำจาก caption ไม่ใช่ field แยกจาก TikTok
- เข้าจากมือถือได้เฉพาะตอนอยู่ในวง Wi-Fi เดียวกับเครื่องที่รันระบบ

## หน้า/ฟีเจอร์

### 1. หน้าเชื่อมต่อบัญชี (/connect)
- แสดงเมื่อยังไม่เคยเชื่อมต่อ หรือ token หมดอายุ
- ปุ่ม "เชื่อมต่อบัญชี TikTok" → ไป TikTok OAuth → กลับมาที่ Dashboard

### 2. หน้า Dashboard (/dashboard) — หน้าแรกที่เห็นทุกครั้งที่เปิดระบบ
- การ์ดสรุป: ยอดผู้ติดตาม, ยอดไลก์รวม, จำนวนคลิป — พร้อมตัวเลข delta เทียบ 7/30 วัน
- กราฟแนวโน้มยอดวิวรวมย้อนหลัง (สลับ 7 วัน / 30 วัน)
- รายการคลิปล่าสุด แต่ละคลิปแสดง thumbnail, ยอดวิว/ไลก์/คอมเมนต์/แชร์, sparkline เล็กๆ
- กดคลิปเพื่อไปหน้ารายละเอียด

### 3. หน้ารายละเอียดคลิป (/clips/[id])
- Thumbnail/embed, caption เต็ม (ไม่แปล), แฮชแท็กที่ตัดคำได้, วันที่ลง, ความยาวคลิป
- ตัวเลขหลัก: วิว ไลก์ คอมเมนต์ แชร์ (ปัจจุบัน + delta)
- กราฟ time-series ของคลิปนี้ (สลับ 7/30 วัน)
- ปุ่ม "บันทึกคลิปนี้" (bookmark toggle)

### 4. หน้าคลิปที่บันทึกไว้ (/bookmarks)
- แสดงคลิปที่กดบันทึกไว้ เรียงจากบันทึกล่าสุด
- การ์ดแบบเดียวกับ Dashboard

### 5. หน้าตั้งค่า (/settings)
- ข้อมูลบัญชีที่เชื่อมต่ออยู่ + ปุ่มยกเลิกการเชื่อมต่อ
- ปุ่ม "ซิงค์ตอนนี้" + เวลา sync ล่าสุด
- ประวัติการ sync ล่าสุด (สำเร็จ/ล้มเหลว)
