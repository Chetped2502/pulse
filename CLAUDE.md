# Pulse — Hard Rules for Claude Code

## สี
- โทนหลัก: ขาว/ดำเท่านั้น (background/foreground แบบ minimal, รองรับ dark mode)
- สีเน้น 1 สีเท่านั้น: coral/pink `#FF3B5C` — ใช้เฉพาะปุ่มหลัก, ลิงก์สำคัญ, จุดไฮไลท์
- สี semantic เพิ่มได้เฉพาะ 2 สี: เขียว (ตัวเลขเพิ่มขึ้น) / แดง (ตัวเลขลดลง) —
  ใช้เฉพาะกับตัวเลข delta ของสถิติเท่านั้น ห้ามใช้ที่อื่น
- ห้ามเพิ่มสีอื่นนอกเหนือจากนี้ในระบบ

## ฟอนต์
- ใช้ sans-serif เดียวทั้งระบบ (เช่น Inter สำหรับอังกฤษ + Noto Sans Thai สำหรับไทย)
- ห้ามผสมหลายฟอนต์ในหน้าเดียวกัน

## ภาษา
- เมนู ปุ่ม label ทั้งหมด: ภาษาไทย
- ข้อมูลตัวเลข ชื่อคลิป caption แฮชแท็ก: ภาษาต้นฉบับตามที่ TikTok ส่งมา
- ห้ามแปล caption หรือชื่อคลิปเป็นไทย

## สิ่งที่ห้ามทำ
- ห้ามใส่สีสันหวือหวาเกิน 1 accent + 2 semantic colors ที่กำหนดไว้
- ห้ามใช้อีโมจิในอินเทอร์เฟส
- ห้ามมี pop-up/modal ที่ไม่จำเป็นต่อ flow หลัก
- ห้ามทำ gamification (badge, level, streak) — ไม่ตรงกับโทนมินิมอล
- ห้ามเก็บหรือแสดงข้อมูลบัญชี TikTok อื่นนอกจากบัญชีของเจ้าของระบบ (ข้อจำกัดของ Official API)
- ห้ามแสดงข้อมูล sound/music ของคลิป เพราะ TikTok Official API ไม่มีข้อมูลนี้ให้ —
  ถ้า field ว่าง ให้ซ่อนไปเลย ไม่ต้องแสดงว่า "ไม่มีข้อมูล"
- ห้ามเพิ่มระบบ login/role หลายคน — ระบบนี้มีผู้ใช้คนเดียวเสมอ
